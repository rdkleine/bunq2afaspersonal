Bunq to Afas Personal

Setting up the service

On this page I will try to explain all the steps necessary to succesfully prepare the Bunq to Afas Personal service to work

Step 1: Generate a RSA key

A 2048 bit RSA certificate is needed to communicate with the Bunq API.

Run the application ssh-keygen to generate a new certificate.

ssh-keygen -t rsa -b 2048

When the certificate is successfully generated it should result in two files. One with a public key and a private key.

The cert file with the private key should never be shared!
